## ✨ امکانات

### 🔐 امنیت و احراز هویت
- **سیستم ورود امن** با رمز عبور هش‌شده (SHA-256 + Salt)
- **سشن کوکی** با اعتبار ۷ روز (HttpOnly)
- **تغییر رمز عبور** از طریق پنل
- **UUID Auth سخت‌گیرانه** برای اتصالات

### 👑 مدیریت کاربران و دسترسی
- **تنها یک ادمین اصلی** با کنترل کامل
- **امکان ساخت کاربران فرعی** با سهمیه حجمی جداگانه
- **محدودیت دستگاه همزمان** (Device Limit) برای هر کانفیگ
- **سیستم سهمیه‌بندی** ترافیک برای هر کاربر

### 🔗 مدیریت کانفیگ‌ها
- **ساخت کانفیگ تصادفی** با UUID یکتا
- **پشتیبانی از پروتکل‌های**: 
  - VLESS + WebSocket (پایدار و همه‌منظوره)
  - XHTTP Ultra (packet-up / stream-up / stream-one)
- **سهمیه ترافیک** (GB/MB)
- **تاریخ انقضا** برای هر کانفیگ
- **فعال/غیرفعال کردن** کانفیگ‌ها
- **محدودیت دستگاه همزمان** (۰ تا ۱۰ دستگاه)
- **فینگرپرینت (Fingerprint)**: Chrome, Firefox, Safari, Edge, Random, None

### 📁 گروه‌های ساب (Subscription Groups)
- **ساخت گروه** برای دسته‌بندی کانفیگ‌ها
- **رمزگذاری صفحه پابلیک** با رمز دلخواه
- **لینک سابسکریپشن** مخصوص هر گروه
- **صفحه پابلیک** برای نمایش کانفیگ‌ها به کاربران
- **QR Code** برای هر گروه و کانفیگ

### 📊 داشبورد حرفه‌ای
- **نمایش اتصالات فعال** (زنده)
- **مشاهده مصرف ترافیک** لحظه‌ای
- **نمودار ترافیک ساعتی**
- **نمودار دایره‌ای توزیع پروتکل‌ها**
- **نمایش وضعیت سرویس‌ها**
- **آپتایم سرور**

### 🖥️ اتصالات زنده
- **نمایش آی‌پی کلاینت‌ها** (با پشتیبانی از پراکسی)
- **مشاهده مصرف هر اتصال**
- **مدت زمان اتصال**
- **گروه‌بندی بر اساس آی‌پی**
- **بروزرسانی خودکار** هر ۵ ثانیه

### 🎨 ظاهر و تم‌ها
- **۷ تم مختلف** (طلایی، بنفش، آبی، مشکی - تاریک و روشن)
- **اسکن لاین متحرک** با قابلیت خاموش/روشن
- **کارت‌های شیشه‌ای (Glassmorphism)**
- **اسکرول‌بار سفارشی**
- **انیمیشن آیکون‌ها** با هاور
- **گرادیانت متحرک دکمه‌ها**
- **پشتیبانی از فونت وزیرمتن**

### 📝 لاگ و خطاها
- **ثبت کامل فعالیت‌ها** (ساخت، ویرایش، حذف کانفیگ و...)
- **نمایش خطاهای رخ‌داده**
- **فیلتر لاگ** بر اساس نوع رویداد

### 🔧 تنظیمات
- **اطلاعات سرور** (پورت، نسخه، فریم‌ورک، پلتفرم)
- **تغییر رمز عبور** با تأیید رمز فعلی
- **مدیریت کاربران فرعی**
- **بخش پشتیبانی** با لینک گروه تلگرام

### 🌐 سایر قابلیت‌ها
- **تست WebSocket** برای عیب‌یابی
- **HTTP Proxy** داخلی
- **سابسکریپشن کامل** از همه کانفیگ‌ها
- **صفحه پابلیک** برای نمایش به کاربران عادی
- **API کامل** برای تمام عملیات‌ها
